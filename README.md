#VTAPE

使用 Flutter 开发的一款在线视频点播 App

目前实现功能:

1.分类列表

2.视频搜索

3.播放记录

4.视频页面播放, 全屏播放

5.由于登录需要验证插件,所以并没有实现, 可自行更新 Token

6.App 内提供了输入 Token 更新功能

注: iOS 视频播放需使用真机



2019.9.28:

目前接口已经被墙, 正常访问需准备梯子

---

使用到的非官方组件:

[fijkplayer](https://github.com/befovy/fijkplayer) 视频播放

pull_to_refresh 列表刷新

ovprogresshud  提示框
